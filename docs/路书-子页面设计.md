# 路书子页面设计

- 入口：发现页顶部分类中的“路书”，右上角提供“我的路书”按钮
- 顶栏：标题“路书”，左侧返回
- 筛选：顶部水平滚动 FilterChip（热门、周边、长距离、爬坡、休闲），对应筛选规则：
  - 热门：全部
  - 周边：按地点包含“上海”示例
  - 长距离：距离≥50km
  - 爬坡：爬升≥500m
  - 休闲：标签包含“休闲”
- 列表卡片：封面图、标题、距离/爬升/地点、难度标签、导航/下载GPX操作、标签
- 数据：使用本地模拟数据；可后续接入后端接口或本地 GPX 解析
- 交互：
  - 筛选切换更新所选类别并过滤列表
  - 点击返回返回到发现主视图
  - 我的路书：顶部 Tab 包含“已下载”“我制作的”“我的收藏”
  - 操作按钮预留回调
