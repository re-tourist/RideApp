# FlowRide 项目结构概览

- 根模块：Android Gradle（Kotlin + Jetpack Compose）
- 主要模块：`app`
- 关键目录：
  - `app/src/main/java/com/example/rideflow/`
    - `auth/` 认证状态与视图模型
    - `di/` 依赖注入模块
    - `model/` 基础数据模型
    - `navigation/` 应用导航图与路由
    - `services/` 后台服务（骑行跟踪）
    - `ui/` 界面层
      - `navigation/` 底部导航相关
      - `screens/` 页面组件（`DiscoverScreen`、`CommunityScreen` 等）
      - `theme/` 主题、颜色与字号
    - `MainActivity.kt` 应用入口
  - `app/src/main/res/` 资源文件（`drawable/`、`values/` 等）
  - `gradle/`、`build.gradle.kts`、`settings.gradle.kts` 构建配置

- 页面结构：
  - 顶层导航由 `AppNavGraph` 管理：登录、注册、主页面等
  - 主页面 `MainScreen` 使用本地状态管理底部四个 Tab：`运动`、`发现`、`社区`、`我的`
  - 子页面在 `ui/screens` 下以独立 `Composable` 实现

- 交互与路由：
  - 顶层路由常量定义于 `navigation/AppRoutes.kt`
  - 主页面内部通过 `MainScreen` 的本地状态切换 Tab，而非嵌套 NavController

- 主题与组件：
  - 使用 Material3 组件库
  - 采用 `Card`、`LazyColumn` 等实现列表与卡片式 UI
