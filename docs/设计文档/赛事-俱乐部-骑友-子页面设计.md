# 发现页子页面设计：赛事 / 俱乐部 / 骑友

- 赛事
  - 入口：发现页“赛事”分类
  - 顶栏：标题“赛事”，左侧返回
  - 筛选：顶部 FilterChip（我的赛事、骑行、越野跑、徒步）
  - 列表卡片：海报、标题、时间、地点、标签、报名入口与报名中按钮

- 俱乐部
  - 入口：发现页“俱乐部”分类
  - 顶栏：标题“俱乐部”，左侧返回，右侧添加与设置按钮
  - 顶部提示：蓝色提示条“你还没有设置主俱乐部，请前往进行设置！”
  - 搜索：输入框“搜索编号、名称、关键字”
  - 分区：本地热门俱乐部、附近俱乐部、全国热门俱乐部，支持“查看更多”入口
  - 列表项：logo、名称、城市、人数、热度（奖杯 + x数值）

- 骑友
  - 入口：发现页“骑友”分类
  - 顶栏：标题“骑友”，左侧返回
  - 标签：顶部 Tab（附近、我的）
  - 列表项：头像、昵称、城市、等级

- 数据与行为
  - 以上页面使用本地模拟数据；筛选与标签切换生效，按钮回调预留。
